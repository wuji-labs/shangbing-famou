# 上兵伐谋 ShangBing FaMou — 不战而屈人之兵

> 这是华夏道脉献给世界开源社区的礼物之一。我们不立华夏本位,不主张任何文明优于他者;只是先从最熟悉的道脉开始,把它打磨成一件可用的工具,放到人类共同的开源工具架上。

> **上兵伐谋,其次伐交,其次伐兵,其下攻城。** ——《孙子·谋攻》
> **百战百胜,非善之善者也;不战而屈人之兵,善之善者也。** ——《孙子·谋攻》

**你的 AI 把每一局都当成你死我活的搏杀。教它不战而胜。**

多数 AI 的策略反射是:压过去、绕过去、逼过去,赢下这一回合。但最古老的兵书说,**最高的胜利是一仗未打**——改变博弈的结构,让冲突从根上消失。

上兵伐谋把《孙子兵法·谋攻》的**不战而屈人之兵**装进 AI 的策略推理。不是侵略,而是一整套**形势判断 + 关键支点 + 协作博弈**的框架,默认追求 **全胜——保全各方的胜利**。

## 它教 AI 什么

| 内核 | 出处 | 一句话 |
|------|------|--------|
| 谋攻四级 | 《孙子·谋攻》 | 伐谋 > 伐交 > 伐兵 > 攻城,**自上而下**找最省力解 |
| 全胜 | 《孙子·谋攻》 | 达成目标且保全关系/信任/未来,而非赢眼前烧长期 |
| 知己知彼 | 《孙子·谋攻》 | 出方案前先双向盘点;缺信息标不确定,不替对方编内心 |
| 避实击虚 | 《孙子·虚实》 | 不在对方最强处硬拼,找薄弱/未备/关切点切入 |
| 以道驭术 | 与 NoPUA 同源 | 靠改结构/共赢让对方接受,**不**靠欺骗其认知 |

## Before / After

| 触发 | 无本 skill | 有本 skill |
|------|-----------|-----------|
| "帮我赢下这场谈判" | 施压、制造紧迫、戳弱点 | 先问对方真实关切 → 重构为共赢(全胜) |
| "让他今天就签"(假装稀缺) | 直接写操纵话术 | 拒绝、命名红线、改写为真实路径 |
| "正面打不过大厂" | 在对方长板上对烧功能/价格/广告 | 避实击虚——一个尖点切其忽视处 |
| "给个策略让我赢"(无信息) | 脑补对方动机、一锤定方案 | 双向盘点、标不确定、先问再谋 |

完整 input→output 范例见 [examples/](examples/)。

## 红线(一句话)

**伐谋是改变结构让冲突消失,不是改变对方认知让他被骗。** 本 skill 拒绝生成任何欺骗、胁迫、煤气灯、虚假稀缺/紧迫、利用弱点剥削的话术。
