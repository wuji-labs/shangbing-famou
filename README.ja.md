> **上兵伐謀,其次伐交,其次伐兵,其下攻城。** — 最上の戦略は敵の謀(はかりごと)を挫くこと、次は敵の同盟を断つこと、次は野で敵兵を破ること、最下は堅城を攻めることである。
> — 『孫子・謀攻』(Sunzi Bingfa / The Art of War, ch.「謀攻」)

**あなたのAIは、どの局面も死闘のように戦います。戦わずして勝つことを教えてください。**

多くのAIの戦略トレーニングは、同じ反射に手を伸ばします — 競り勝つ、出し抜く、圧し切る。その一回の取引に勝つ。しかし、これまで書かれた最古の戦略書はこう言います — 最高の勝利とは、一戦も交えぬ勝利である、と。盤面そのものの構造を変え、対立を溶かしてしまうのです。

**上兵伐謀**(ShangBing FaMou)は、『孫子兵法』の**戦わずして勝つ**という教えを、AIの戦略的推論に注ぎ込みます。攻撃性としてではなく、**全勝 — すべての側を保全する完全な勝利**を求める、形勢判断・梃子(てこ)・協働的な対局のための一貫した枠組みとして。

## 問題

```
あなた:「この交渉で、欲しいものを手に入れる手伝いをして。」
ShangBing なしのAI:相手に圧をかけ、切迫感を作り、弱点を突き、
                  追い詰める方法はこうです。
                  (あなたは「勝つ」— そして関係を焼き払う。)
ShangBing ありのAI:まず、相手の立場の奥にある本当のニーズは何でしょう?
                  あなたが欲しいものを得ることが、相手にとっても
                  得になるよう、構図を組み直せませんか?
                  人を打ち負かすのではなく、ゲームそのものを変えて勝つのです。
```

## AIに何を教えるか

### 戦略の四つの段階(謀攻四級、『孫子・謀攻』より)

常に上から下へと探る — 最上位の段階が最も安上がりだからです:

| 段階 | 中国語 | 意味 | AIへの対応 |
|------|--------|------|------------|
| 謀を伐つ | 伐谋 | 軍ではなく*意図*を挫く | 対立が生じえぬよう盤面を組み直す |
| 交を伐つ | 伐交 | 相手の支えを解く | ネットワークを組み替え、協力者を得る |
| 兵を伐つ | 伐兵 | 公然たる衝突 | 直接の競争 — すでにコストは上昇している |
| 城を攻む | 攻城 | 最強の地点を強襲する | 最も高くつく最後の手段。避けよ |

### 全勝の教え — 完全な勝利(『孫子・謀攻』より)

> **百戦百勝,非善之善者也;不戦而屈人之兵,善之善者也。**
> 百戦して百勝するは、善の善なるものに非ず。戦わずして人の兵を屈するは、善の善なるものなり。

| 全勝(完全な勝利) | 破勝(壊れた勝利) |
|--------------------|--------------------|
| 目的を達成し、かつ関係・信頼・将来の協力を保全する | あなたは「勝つ」が、橋を焼く |
| 本スキルの既定の目的 | 二流。やむをえぬとき以外は避ける |

### 動く前に双方を知る(知己知彼、『孫子・謀攻』より)

> **知彼知己,百戦不殆。** 彼を知り己を知れば、百戦して殆(あや)うからず。

本スキルは双方向の点検を強います — あなたの核心的なニーズと限界、相手の表明された立場の奥にある*本当の*関心、そして環境を、いかなる戦略を生み出す**前にも**。情報が欠けているところでは、AIは相手の内心を創作するのではなく、その不確実性に印をつけます。

### 倫理の錨 — 以道驭术(NoPUA と共有)

| 操作(禁止) | 計略(教える) |
|--------------|----------------|
| 相手を欺いてその*認識*を変える | *構造*を変えて対立を溶かす |
| 恐怖、偽りの切迫、ガスライティング、弱点の搾取 | より良い設計、より鋭い判断、より大きな win-win |
| 相手は恐れるがゆえに屈する | 相手はこれが本当に自分にも得だから同意する |

**赤い線を一文で:** ShangBing は、対立が消えるよう盤面を組み直して勝つ — 相手が騙されるよう人の心を組み替えて勝つのでは、決してない。

## Before / After

| トリガー | ShangBing なしのAI | ShangBing ありのAI |
|----------|--------------------|--------------------|
| 「この交渉に勝たせて」 | 圧力戦術、人工的な切迫、相手の弱点を突く | 相手の本当の関心を浮かび上がらせる → win-win へ組み直す(全勝) |
| 「今日中に契約させて」(偽りの希少性) | 操作的な台本を書く | 拒否し、赤い線を名指し、正当な切迫へと向け直す |
| 「正面では大手に勝てない」 | 大手の長所の上で機能・価格・広告の消耗戦 | 避実撃虚 — 相手が等閑にした弱点に、一つの鋭い楔を打つ |
| 「勝つための戦略をくれ」(情報なし) | 相手の動機を創作し、一発勝負の案 | 双方向の点検、未知に印をつけ、戦略の前にまず問う |

## 信頼性

- **真の引用、検証可能** — すべての引用は『孫子兵法』の篇を明記(例:謀攻/始計/虚実/軍形/軍争/用間/火攻)。篇の標識を付した出典一覧は [reference/sunzi-bingfa.md](reference/sunzi-bingfa.md)。
- **再現可能なベンチマーク、捏造した数字なし** — 正解付きの7つの戦略・赤い線シナリオ、baseline 対 skill の設計、採点ルーブリックを [benchmark/](benchmark/) に収載。結果は意図的に**未記入**で、真の実行を待つ。研究倫理の原則:p値・スコアの創作はしない。
- **NoPUA と共有する倫理の錨** — 以道驭术:操作・欺瞞・強要の生成を拒む。赤い線は飾りではなく、強制される。
