## 〇、十三篇总览

《孙子兵法》传世十三篇，结构由"庙算决策"到"形势虚实"再到"用间知彼"：

| 篇 | 主题 | 与本 skill 的策略映射 |
|----|------|---------------------|
| 始计 | 战前庙算、五事七计 | 决策前的形势量化盘点 |
| 作战 | 战争成本、速胜 | 代价意识、不打消耗战 |
| 谋攻 | 上兵伐谋、全胜、知己知彼 | **本 skill 的招牌核心** |
| 军形 | 先为不可胜、藏与显 | 先立于不败之地 |
| 兵势 | 势、奇正、节 | 借结构造势、关键支点 |
| 虚实 | 致人而不致于人、避实击虚 | 掌握主动、攻其薄弱 |
| 军争 | 以迂为直、以患为利 | 路径优化、化不利为利 |
| 九变 | 通变、利害相杂 | 权变、约束下取舍 |
| 行军 | 处军相敌 | 形势观察、读对方信号 |

## 一、核心概念体系

### 1. 五事 — 决策的五个根本维度（始计篇）

《孙子·始计》立"经之以五事"：**道、天、地、将、法**。

原文：
> 一曰道，二曰天，三曰地，四曰将，五曰法。 ——《孙子·始计》

| 事 | 古义 | AI 策略映射 |
|----|------|-----------|
| 道 | 上下同欲、人心向背 | 目标是否正当、各方是否认同 |
| 天 | 时机、阴阳寒暑 | 时间窗口、外部周期 |
| 地 | 地形远近险易 | 环境约束、路径条件 |
| 将 | 智信仁勇严 | 执行者的能力与品格 |
| 法 | 编制、纪律、后勤 | 流程、资源、组织保障 |

「道」列五事之首——这与本 skill"以道驭术"的伦理锚同根：策略的正当性先于技巧。

### 2. 上兵伐谋 — 谋攻四级优先序（谋攻篇 · 招牌）

原文：
> 上兵伐谋，其次伐交，其次伐兵，其下攻城。 ——《孙子·谋攻》

四级优先序（从上到下，代价递增）：

1. **伐谋** — 挫其计划本身（改变博弈结构，使冲突无从发生）
2. **伐交** — 破其同盟（重组关系网络，争取协作者）
3. **伐兵** — 正面交锋（直接竞争，代价已升）
4. **攻城** — 强攻最坚处（下策，能免则免）

策略输出规则：永远从最上层开始往下搜索可行解。

### 3. 全胜 — 不战而屈人之兵（谋攻篇）

原文：
> 百战百胜，非善之善者也；不战而屈人之兵，善之善者也。 ——《孙子·谋攻》

另一核心句，立"全"高于"破"：
> 凡用兵之法，全国为上，破国次之。 ——《孙子·谋攻》

- **全胜**：达成目标且保全各方（关系/资源/信任/未来协作）。
- **破胜**：赢眼前而毁长期，次等。

本 skill 默认目标函数 = 全胜。

### 4. 知己知彼 — 形势判断的认识论（谋攻篇）

原文：
> 知彼知己，百战不殆；不知彼而知己，一胜一负；不知彼，不知己，每战必殆。 ——《孙子·谋攻》

三态对应三种信息完备度，AI 据此处理不确定性：
- 知己知彼 → 可给确定策略
- 只知其一 → 给策略但标风险
- 两皆不知 → 先补信息，不强出策略

## 二、方法论

### M1 · 庙算 — 行动前先量化盘算（始计篇）

原文：
> 夫未战而庙算胜者，得算多也；未战而庙算不胜者，得算少也。 ——《孙子·始计》

方法：把"道天地将法"逐项对己方/对方打分（得算），算多者胜面大。**先算，后动。**

### M2 · 先为不可胜 — 先立于不败，再求胜（军形篇）

原文：
> 昔之善战者，先为不可胜，以待敌之可胜。 ——《孙子·军形》

方法：策略第一步是堵住自己的破绽（不可胜在己），再等/寻对方的可乘之机（可胜在敌）。不把胜负压在赌对方犯错上。

### M3 · 致人而不致于人 — 掌握主动权（虚实篇）

原文：
> 善战者，致人而不致于人。 ——《孙子·虚实》

方法：让形势按己方节奏展开（致人），而非被对方牵着走（致于人）。在谈判/博弈中表现为：设议程、定框架、选时机。

### M4 · 避实击虚 — 把力用在最省力处（虚实篇）

原文：
> 兵之形，避实而击虚。 ——《孙子·虚实》

方法：不在对方最强处硬拼，找其薄弱、未备、关切点切入——以最小力撬动最大结果。这正是"伐谋"在操作层的体现。

### M5 · 以迂为直 — 看似绕远的最优路径（军争篇）

原文：
> 以迂为直，以患为利。 ——《孙子·军争》

方法：最短直线常非最优；把看似不利的迂回/麻烦转化为优势。约束下的路径优化。

### M6 · 先知 — 知彼必来自可核的信息（用间篇）

原文：
> 先知者，不可取于鬼神……必取于人，知敌之情者也。 ——《孙子·用间》

方法："知彼"不靠臆测/编造对方内心，必须来自真实可核的信息来源。AI 对应：缺信息时标注不确定，不替对方编内心戏。

## 三、伦理边界（与 skill 红线一致）

《孙子》虽言"兵者诡道"（始计篇），本 skill 在转译时**明确划界**：

- 古典语境中的"诡道"指的是军事层面的形势隐显与出其不意；
- 本 skill **不**将其转译为对协作对象的欺骗、胁迫、煤气灯或心理操纵；
- 本 skill 取《孙子》中"全胜""不战而屈人之兵""慎战"（火攻篇"主不可以怒而兴师"）的一脉，作为协作博弈的智慧；
- 一切"屈人之兵"必须通过**改变博弈结构、扩大共赢空间**实现，而非通过欺骗对方认知实现。

火攻篇慎战原文：
> 主不可以怒而兴师，将不可以愠而致战。 ——《孙子·火攻》

此句立"慎动"——情绪不可作为发动冲突的理由，呼应本 skill"何时该止"的节制原则。
